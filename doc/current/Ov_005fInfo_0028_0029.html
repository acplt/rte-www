5.1 Print debug information

If OV_DEBUG is defined, this macro logs a user-defined debugging message:

[ACPLT/OV Debug] file:line: Info: info

Header file

#include "libov/ov_debug.h"

Macro usage

Ov_Info(info);

Parameters

info
String containing the info message.

Return value

none
