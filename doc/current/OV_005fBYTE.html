7.21 Generic byte value

A byte value is a generic value which can store 8 bits.

Header file

#include "libov/ov_ov.h"

Declaration

typedef char OV_BYTE;

Remarks

Byte values are not used in ACPLT/OV; however, opaque values are expressed as byte vectors in ACPLT/KS. In ACPLT/OV this feature is used to allow to generically transport variable values over ACPLT/KS which are not supported by ACPLT/KS (e.g. user-defined datatypes).

Furthermore, sometimes it is necessary to calculate the difference between to pointers in bytes. As ANSI C does not define a difference of void* pointers, for this purpose OV_BYTE* pointers are used.
